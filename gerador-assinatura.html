Como usar no seu e-mail

  1. Clique em "Copiar para Gmail / Outlook" acima
  2. No Gmail: Configurações → Ver todas → Role até "Assinatura" → Cole com Ctrl+V ou Cmd+V
  3. No Outlook: Novo email → Inserir → Assinatura → Cole o HTML copiado
  4. No Apple Mail: Preferências → Assinaturas → Cole o conteúdo
  5. Se precisar do HTML puro (para outros clientes), use "Copiar HTML da Assinatura"
